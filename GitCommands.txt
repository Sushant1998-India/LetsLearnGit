Git Commands

git --version // to check the version of Git

git config --global user.name "Sushant"  // to set the username Sushant

git config --global user.name "sushantshastri24@gmail.com" //to set the user email

git config --global --edit // you can edit the user name and email from this command too

CTRL+:+WQ // to return backinto terminal

mkdir // to make directory

cd:// to locate current directory

git init // to create repository into directory

git status// to check the status off the files

Following is the process followed by git:
 Working Directory: Making changes in the file

 git add <filename>: After executing this command respective file comes in staging area

 git commit -m <message> After executing this command respective files comes under repository

git log  // to check the logs of files

git branch <branchname> to add branches to main branch

git checkout <branchname> to go to that branch

git push
